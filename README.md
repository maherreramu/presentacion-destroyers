# presentacion-destroyers

Presentación estática lista para GitHub Pages.

## Despliegue

1. Sube este repositorio a GitHub.
2. En Settings > Pages, selecciona GitHub Actions como fuente.
3. Cada push a `main` publicará el contenido del repositorio en Pages.

## Estructura

- `index.html`: presentación principal.
- `charts/`: imágenes usadas por las diapositivas.
- `.github/workflows/pages.yml`: despliegue automático a GitHub Pages.
